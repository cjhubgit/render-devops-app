Project Purpose

The purpose of this project is to showcase my understanding of modern DevOps workflow skills by building and automating the deployment (CI/CD) pipeline. This project demonstrates my ability to:

  • Use version control with Git and GitHub.
  • Create and deploy a Python application with Flask.
  • Automate deployment using GitHub Actions and Render's hosting platform.
  • Implement best practices in DevOps to ensure consistent and scalable workflows.
